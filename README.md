# 🏪 Dự án App ATM Python Cơ Bản

Hi mọi người, đây là dự án đầu tay của mình được viết hoàn toàn bằng hàm `def` trong Python vào năm lớp 8!

## 🚀 Chức năng của App:
* Đăng ký tài khoản mới (Check độ dài tài khoản, mật khẩu, chống khoảng trắng).
* Đăng nhập hệ thống bảo mật.
* Xem số dư tài khoản hiện tại.
* Nạp tiền và rút tiền (Có check điều kiện số dư, chống nhập chữ bằng try-except).
